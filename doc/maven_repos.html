Terrier Maven Repository

To download the lastest version of Terrier through Terrier Maven Repository, add the following Maven configuration in your pom.xml file: 


<repositories> 
  <repository>
    <id>terrier.snapshot</id>
    <url>http://demos.terrier.org/artifactory/libs-snapshot-local/</url>
  </repository>
</repositories>
  
<dependencies>
  <dependency>
    <groupId>org.terrier</groupId>
    <artifactId>terrier-core</artifactId>
    <version>4.1-SNAPSHOT</version>
  </dependency>
</dependencies>
